ОТМЕТИЛИ

Транзакции - нужен поиск как папки, так и файла (либо один документ с точным номером, либо папка целиком)

Номер транзакции может повторяться с 1 апреля каждого года (старт финансового года в САП). В рамках одного месяца, даже одного финансового года, повторов номеров не будет.

Платежки искать внутри папки разархивированной. Копируется только файл

Для инвойсов логика: Робот смотрит в каждую подпапку инвойса и ищет конкретные файлы

80% запросов ожидается на транзакции, 15% на платежки, 5% на инвойсы - поэтому длительный поиск инвойсов не должен быть проблемой

Для инвойсов нужно искать CMR, GTD, AVB (должно быть по одному файлу) сам инвойс - INV (мб несколько), все файлы должны быть приоритетно пдф. Сотрудники будут переименовывать - начиная с 2025. Также может быть Транзитная декларация
